# 😂 わらいのツボ診断

あなたの「笑いのクセ」を4タイプに分類するWeb診断ツールです。
全20問・5段階選択式。スマホでそのまま動作します。

## 🎭 診断できる4タイプ

| タイプ | 特徴 |
|--------|------|
| 🌸 ほっこり共感タイプ | 日常あるある・誰も傷つかない笑い |
| 💡 ひらめき構造タイプ | 言葉遊び・伏線回収・ネタ構造の理解 |
| 🌶 ピリ辛スパイスタイプ | ブラックジョーク・緊張と解放のバランス |
| 🌀 ふしぎカオスタイプ | 不条理・意味不明・説明できない面白さ |

## 🚀 使い方

### そのまま開く
```
index.html をブラウザで開くだけでOK
```

### GitHub Pages で公開する場合
1. このリポジトリを fork またはクローン
2. `Settings` → `Pages` → `Branch: main` / `/ (root)` を選択
3. 公開URLが発行されます（例：`https://yourname.github.io/warainotsubo/`）

## 📁 ファイル構成

```
/
├── index.html   # 診断ツール本体（全機能を1ファイルに集約）
└── README.md
```

> 外部依存はGoogle Fonts（Zen Marugothic / Kaisei Decol / DotGothic16）のみ。
> ライブラリ・フレームワーク不要。

## ⚙️ 技術仕様

- **HTML + CSS + Vanilla JS** の単一ファイル構成
- 質問は配列で管理、毎回ランダムシャッフル
- 各タイプに5問ずつ対応（合計20問）
- 5段階評価（5〜1点）でスコア集計、最高点タイプを表示
- 同点時はサブタイプバッジ＋補足テキストを表示
- SNSシェア用テキストをワンタップでコピー可能

## 🎨 デザイン

- パステル調・明るめカラー
- Kaisei Decol（見出し）× Zen Marugothic（本文）
- 1問ずつスライド表示＋進捗バー
- 選択時の絵文字フローティングエフェクト
- 結果のスコアバーアニメーション
