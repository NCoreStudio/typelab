# 診断ツール追加方法

新しい診断ツールをサイトに追加する方法は簡単です。以下の手順で自動的にサイト全体が更新されます。

## 🚀 簡単追加方法

### 1. 診断ツールフォルダを追加
サイトのルートに新しい診断ツールのフォルダを追加します。

例：`新しい診断/` フォルダを作成
```
総合サイト/
├── 新しい診断/
│   ├── index.html
│   ├── style.css
│   └── app.js
└── index.html
```

### 2. 設定ファイルに追加
`diagnosis-config.js` を開き、新しい診断の情報を追加します。

```javascript
{
  id: 'new-diagnosis',                    // 一意のID
  title: '新しい診断ツール',              // 表示タイトル
  desc: '新しい診断の説明文',             // 説明文
  emoji: '🆕',                           // 絵文字
  genre: 'friend',                        // ジャンル: love/friend/work/other
  genreLabel: '👫 友達関係',              // ジャンルラベル
  path: './新しい診断/index.html',        // ファイルパス
  gradient: 'linear-gradient(135deg, #d9f5ec 0%, #e8f7fc 100%)', // 背景グラデーション
  addedDate: new Date()                  // 追加日時（自動設定）
}
```

### 3. 自動更新
これだけで完了です！サイトをリロードすると、以下が自動的に更新されます：

✅ **新着診断セクション**：最新3件が表示され、新しい診断が先頭に表示されます  
✅ **ジャンル別セクション**：該当ジャンルに自動的に追加されます  
✅ **フッター**：最新の診断リンクが更新されます  
✅ **統計情報**：公開中の診断数が自動的にカウントアップされます  

## 📋 ジャンル一覧

| ジャンル | ID | ラベル | 色 |
|---------|----|----|----|
| 恋愛関係 | `love` | `💕 恋愛関係` | ピンク系 |
| 友達関係 | `friend` | `👫 友達関係` | 緑系 |
| 仕事関係 | `work` | `💼 仕事関係` | 青系 |
| その他 | `other` | `🌈 その他` | オレンジ系 |

## 🎨 グラデーション例

各ジャンルのおすすめグラデーション：

```css
/* 恋愛関係 */
linear-gradient(135deg, #ffe4ef 0%, #fde8e8 100%)

/* 友達関係 */
linear-gradient(135deg, #d9f5ec 0%, #e8f7fc 100%)

/* 仕事関係 */
linear-gradient(135deg, #dde9ff 0%, #ede9fe 100%)

/* その他 */
linear-gradient(135deg, #fef0d5 0%, #ffeef4 100%)
```

## ⚡ 自動更新機能

このシステムにより、以下の手作業が不要になります：

- ❌ HTMLファイルを直接編集する必要なし
- ❌ リンクを手動で更新する必要なし  
- ❌ カードをコピー＆ペーストする必要なし
- ❌ 統計数値を手動で変更する必要なし

## 🔧 カスタマイズ

より詳細なカスタマイズが必要な場合は、`index.html` 内の `DiagnosisManager` クラスを編集してください。
